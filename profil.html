Загальна інформація

Металевий профіль — основний елемент каркасної системи для монтажу гіпсокартонних перегородок, стель, облицювання стін. Виготовляється з оцинкованої сталі методом холодного прокату, що забезпечує високу жорсткість і стійкість до корозії.

Додаткові елементи

  • Підвіс прямий (125, 200 мм) — кріплення CD профілю до базової поверхні.
  • Підвіс анкерний — для регулювання висоти підвісної стелі.
  • Краб-з’єднувач — для з’єднання профілів CD у двох площинах.
  • Подовжувач CD–CD — нарощування профілю по довжині.
  • Шурупи LN / LB — для кріплення гіпсокартону і профілів.
  • Ущільнювальна стрічка — для шумо- і віброізоляції під направляючими.
